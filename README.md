# MindSpark

Production-ready MVP for an AI tutor chat app.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express + Prisma
- Database: PostgreSQL
- Auth: JWT access token + refresh token in httpOnly cookie
- AI provider: OpenAI-compatible API (`agentplatform.ru`) with `mistralai/mistral-nemo` by default

## Local run

1. Start PostgreSQL:
   `docker compose up -d`
2. Install deps:
   `npm install`
3. Configure env:
   copy `apps/api/.env.example` to `apps/api/.env`
4. Push schema:
   `npm run db:push`
5. Start dev servers:
   `npm run dev`

## Render deploy

This repo includes a root-level [render.yaml](./render.yaml) for a Render Blueprint with:

- `mindspark-web` as a Static Site
- `mindspark-api` as a Node Web Service
- `mindspark-db` as Render Postgres

Important notes:

- On first deploy, set `CLIENT_URL` in the API service to your frontend Render URL or custom domain.
- Set `OPENAI_COMPAT_API_KEY` manually in Render.
- `JWT_ACCESS_SECRET` and `JWT_REFRESH_SECRET` are auto-generated by the Blueprint.
- Prisma schema is applied via `preDeployCommand`.
